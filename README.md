# Air Quality Checker

## Overview

This project is a simple web application that allows users to check the air quality of a specific location based on latitude and longitude coordinates. The application fetches air quality data from a third-party API and displays various air quality parameters, such as AQI (Air Quality Index), CO, NO2, O3, PM2.5, PM10, and SO2 readings.

## Features

- Input fields for latitude and longitude coordinates.
- Fetches real-time air quality data using a third-party API.
- Displays the fetched air quality parameters on the webpage.
